# Framez E-commerce application for HNG13 stage 4 Frontend task.

## Technologies used

- [Expo](https://docs.expo.dev/)
- [React native](https://reactnative.dev/)
- [Supabase](https://supabase.com/)
- [React Navigation](https://reactnavigation.org/docs/)
- [Tanstack Query](https://tanstack.com/query/latest)

## Features

- Persisted Credentials Authentication with Logout functionality.
- Create post
- Comment on a post
- Context API for local storage management.
- View post details
- Update personal profile (about me, profile pic, display name)

## Screens

- Auth (Login, Signup)
- Posts (Feed, Add new post, Comment, Post author profile, Post Details)
- Settings
- User Profile
